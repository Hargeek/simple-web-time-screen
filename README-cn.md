一个简单的网页版数字时钟，可用作屏幕保护程序

## 功能特点

- 时间显示
- 日期显示（公历和农历）
- 位置显示（基于IP）
- 深色/浅色主题
- 字体大小调节
- 显示模式切换

## 使用方法

在浏览器中打开 `index.html` 即可使用
